🔵Warm-up⏱ 5–7 min
THIS SESSION · Irregular Snap

Teacher types an infinitive in chat. Student types the past form as fast as possible. If correct — 1 point. If wrong — teacher shows the correct form. Go through all 12 irregular verbs. Then switch: teacher gives past form, student gives the infinitive.

Why: irregular verbs are pure memory — the fastest way to learn them is repetition in a game format.

⚡Wrap-up & Error Correction⏱ 3–5 min
Common errors — A1.2 · Irregular Past Verbs
❌ I goed to the meeting.
→
✅ I went to the meeting.
Go is irregular. There is no 'goed'. go → went — memory only.
❌ Did you went to the conference?
→
✅ Did you go to the conference?
After Did, always use the BASE verb. went is not used after Did.
❌ We maked a great proposal.
→
✅ We made a great proposal.
Make is irregular. make → made — not maked.
